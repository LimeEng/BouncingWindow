# BouncingWindow
A quick and fun experiment to try out Scala and ScalaFX

## SBT help

Run the project
```
sbt run
```

Package the project
```
sbt package && mv target\scala-2.11\bouncing-window_2.11-1.0.jar bouncing-window.jar
```
